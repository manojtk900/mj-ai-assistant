# 🤖 MJ AI Assistant

MJ is a personal AI voice assistant inspired by Jarvis (Iron Man).

## 🚀 Features

- 🎤 Voice Command System
- ⌨️ GUI Typing Support
- 🧠 AI Brain (ChatGPT / Ollama)
- 💾 Memory System
- 👁 Vision AI (Camera)
- 🗣 Text-to-Speech (Female Voice)
- 🧠 Emotion Detection

---

## 🖥️ Demo

> MJ can open apps, answer questions, remember tasks, and talk like a human.
